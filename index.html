About Assocify

Assocify is a Discord app created by talchao that associates messages with songs, gifs, etc.

In servers, Assocify checks messages sent by users for keywords and responds with related links. Additionally, commands such as /song and /media can be run in servers and DMs alike with a limited selection of options. To use Assocify commands in direct messages, the Assocify app must first be added to the user's Discord account. If you would like to install the app: click here, then select "Add to My Apps".
